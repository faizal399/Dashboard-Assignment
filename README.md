# Employee Dashboard React Application

## Features

- Authentication with Protected Routes
- Employee List from API
- Employee Details Page
- Camera Capture Feature
- Photo Result Page
- Salary Bar Graph (Top 10)
- Interactive Map with City Markers

## Technologies Used

- React
- React Router
- Tailwind CSS
- Recharts
- React Leaflet
- React Webcam
